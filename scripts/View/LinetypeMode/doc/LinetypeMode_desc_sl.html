Preklopi način črte trenutne risbe.
Ko je način črte aktiviran, so črte optimirane za računalniški ekran. V tem načinu so prikazane vse črte z debelino enega pixla ter vsi tipi črt so prikazani kot vzorci temelječ na debelini pixsla. Črtkane črte so tako vedno vidne kot črtkane črte.
Ko je način črte izklopljen, so tipi črt prikazani v enotah risbe (privzeta nastavitev).
